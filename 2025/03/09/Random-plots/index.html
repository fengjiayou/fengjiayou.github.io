Nginx配置

server {  
    listen 80;  
    server_name img.yourdomain.com;  

    # 伪装图片路径  
    location = /random.jpg {  
        rewrite ^ /random.html break;  
    }  

    # 静态资源  
    location / {  
        root /var/www/static;  
        index index.html;  
    }  
}

3. 高级技巧

动态生成JSON： 

# 使用脚本自动生成image-list.json  
#!/bin/bash  
IMAGES=($(ls /var/www/images/*.jpg))  
JSON="["  
for img in "${IMAGES[@]}"; do  
    JSON+="\"https://img.yourdomain.com/${img##*/}\","  
done  
JSON="${JSON%,}]"  
echo $JSON > /var/www/static/image-list.json

七、性能优化大全

1. 缓存策略

层级 方案 效果
浏览器 Cache-Control: max-age=86400 减少重复请求
CDN Cloudflare 缓存规则 加速全球访问
服务端 Redis缓存图片列表 降低数据库压力

2. 安全防护

  • 防盗链设置： 

    valid_referers none blocked *.yourdomain.com;  
if ($invalid_referer) { return 403; }

  • 请求限流： 

    limit_req_zone $binary_remote_addr zone=api_limit:10m rate=10r/s;  

location /api {  
    limit_req zone=api_limit burst=20;  
}

八、常见问题解答

Q1：如何解决跨域问题？

  • 后端设置 Access-Control-Allow-Origin 头
  • 使用Cloudflare Workers代理

Q2：图片加载慢怎么办？

  • 启用CDN加速
  • 转换WebP格式（节省30%体积）
  • 实现懒加载技术

Q3：如何防止图片被盗用？

  • 添加动态水印
  • 设置Referer白名单
  • 使用签名URL
